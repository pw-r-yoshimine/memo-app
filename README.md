# memo-app

- ハンズオン用のメモアプリです。
- Jetpack Compose と ViewModel を用いて、MVVM アーキテクチャを学ぶことを目的としています。 

---

## 機能

- メモ一覧の表示
- メモの新規作成
- メモの編集・削除

---

## セットアップ方法

1. **Android Studio を起動**
2. メニューから **[File] → [New] → [Project from Version Control]** を選択
3. 以下のリポジトリ URL を入力
https://github.com/pw-r-yoshimine/memo-app.git
4. クローン後、ビルドが完了したら  ▶️ **Run** ボタンで実行！
